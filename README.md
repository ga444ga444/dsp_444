### 00 · Customer Segmentation (K-Means Clustering)

Applied RFM (Recency, Frequency, Monetary) feature engineering to 541,909 retail transactions, then used K-Means clustering to segment customers into four actionable groups.

**Dataset:** [UCI Online Retail Dataset](https://archive.ics.uci.edu/ml/datasets/online+retail)

| Segment | Share |
|---------|-------|
| VIP | 16.5% |
| Promising | 19.3% |
| Needs Attention | 27.0% |
| Hibernating | 37.2% |
